list-style-position: inside

  • This is a list item that wraps to the next line and the text starts under the bullet.
    Another example using the "inside" setting.

list-style-position: outside

  • This is a list item that wraps to the next line and the text aligns with the start of the first line.
    Another example using the "outside" setting.

outside (default): The bullet or number is placed outside the content box, to the left of the text. If the list item wraps onto multiple lines, the text aligns with the start of the text, not the bullet. inside: The bullet or number is placed inside the content box, as part of the text flow. If the text wraps, it aligns under the bullet, not under the start of the text.
