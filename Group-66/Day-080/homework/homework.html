Explanation of Secured Datas

TCP

TCP makes sure both computers agree to communicate (handshake process), so it can send info between them safely, making sure there are no errors while transferring

TCP (Transmission Control Protocol) ensures reliable communication by establishing a connection through a three-way handshake.
It splits data into packets, numbers them, and checks for errors, guaranteeing that all packets arrive in the correct order.
If a packet is lost, TCP requests a retransmission, providing error detection, flow control, and reliable delivery of data between computers.

TLS

TLS makes sure no one reads the information (man-in-the-middle attacks) while transferring. it encrypts the information (secret key)

TLS (Transport Layer Security) encrypts data to prevent eavesdropping and tampering during transmission.
It uses a combination of symmetric and asymmetric encryption to secure communication.
TLS authenticates the server (and optionally the client) and ensures integrity of the data.
This prevents attackers from intercepting or modifying the transmitted information, providing confidentiality and trust.

HTTP

HTTP is a set of rules for the information (Policy/Format)

HTTP (HyperText Transfer Protocol) defines how messages are formatted and transmitted on the web.
It specifies how clients request data from servers and how servers respond, including status codes, headers, and message bodies.
HTTP is stateless, meaning each request is independent.
It allows retrieving web pages, images, and other resources according to a standard protocol that both clients and servers understand.

HTTPS

HTTPS is the same as HTTP, but it's secure, so no one reads the information and confirms that server is real which we are talking to. (uses TLS)

HTTPS (HyperText Transfer Protocol Secure) combines HTTP with TLS encryption to secure web communications.
It ensures that data sent between the client and server is encrypted, preventing interception and tampering.
HTTPS verifies the authenticity of the server using digital certificates, protecting users from connecting to fraudulent or malicious websites.
It maintains both privacy and data integrity for secure communication.
